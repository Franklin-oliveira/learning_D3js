Interval Loops

Similar to JavaScript setInterval method, D3 has d3.interval attribute which does the exact same thing.

In javascript, the syntax is:

let myInterval = setInterval(function () {
// do something
}, 500)


clearInterval(myInterval) // stops the loop

In D3 it's pretty similar:

d3.interval(function () {
// do something
}, 500)


To do this, we should think which parts of our code should run every time our data changes and which parts should run just once.


D3 Update Pattern

  1. Data Join: select all matching elements on the screen with selectAll and update the data we're using.
  2. Exit: use the exit() selector to remove the elements that don't exist in the new array of data
  3. Update: set attributes for existing elements on the screen
  4. Enter: use the enter() selector to set attributes for new items in the data array.
