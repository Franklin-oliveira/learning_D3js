SVG object

To create a new svg object, you need to use the <svg> tag and set window dimmensions (width and height) to it, like:

<svg width="400px" height="60px"></svg>

p.s.: the (x,y) coordinates start at (0,0) on the top left corner of the screen.

Next, there are some examples of what you can draw in it.

Drawing Rectangles

To draw a rectangle, you can use the <rect > tag. It takes as arguments:

  • x and y coordinates.
  • height and width.
  • fill: color to fill the shape.
  • stroke: color of the contour.
  • stroke-width: width of the contour.

p.s.: the (x,y) coordinates start at (0,0) on the top left corner of the shape object.

Circles

To draw a circle, there is the <circle> method. You need to pass

  • cx and cy, which stands for x and y coordinates of the center.
  • r: radius.
  • fill: color to fill the circle.

Ellipses

To draw an elipse, just go for the <ellipse> method. Similar to the circle, you need to pass the x and y coordinates and, maybe, a fill color. Although, there are two more necessary arguments:

  • rx: radius parallel to the x axis.
  • ry: radius parallel to the y axis.
